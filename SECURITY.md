# Security Policy

## Supported Versions

This repository is maintained on the default branch. Security fixes are applied there first.

## Reporting A Vulnerability

Please do not open a public issue for a suspected security problem.

Instead:

1. Use a private GitHub security advisory if it is available for the repository.
2. If that is not available, contact the maintainer through <https://github.com/gokhanamal>.

When reporting an issue, include:

- A short description of the problem
- Affected files or skills
- Steps to reproduce
- Potential impact
- Any suggested fix or mitigation

You can expect an acknowledgment once the report is seen, followed by triage and a coordinated fix when the report is valid.
